# gerenciadorDeVendas

Este é um Projeto de PDV desenvolvido em Java 8 com Hibernate, Swing e Groovy com foco principal em lojas de roupa. 

Com o uso da base de dados MariaDB embarcada foi possível ter uma base de dados integrada e totalmente multiplataforma sem precisar instalar um banco de dados.

Foi utilizado o JasperReports e outras ferramentas para a impressão de etiquetas
